# Systems By Building

> Learning computer science by building real systems from scratch.

This repository is my long-term engineering journey to deeply understand how modern software systems work internally — not just use them.

Instead of only watching tutorials, I will build the core ideas behind operating systems, databases, networking, compilers, distributed systems, cloud infrastructure, AI frameworks, and more.

---

# Goals

- Learn low-level and high-level system design
- Understand how production software works internally
- Improve problem-solving and engineering skills
- Become capable of building scalable systems from scratch
- Document everything publicly
